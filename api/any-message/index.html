AnyMessage

AnyMessage
typescript
type AnyMessage = AllMessages<TDefs> | AllResults<TDefs>

Union of all message types (requests and responses). Use this for middleware, transaction ID generators, and other functions that handle any message type.

Examples

typescript
type MyMessages = DefineMessages<{
  load: { payload: { config: string }; result: { loaded: true } };
}>;

function middleware(msg: AnyMessage<MyMessages>) {
  console.log('Handling:', msg.type);
}
