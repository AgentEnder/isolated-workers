# AI Context Folder

This folder contains context and guidance for AI agents working on the isolated-workers project.

## Contents

- **plans/**: Implementation plans outlining development phases and tasks
- **design-decisions/**: Architecture Decision Records (ADRs) documenting key design choices

## Purpose

Centralizing AI context in this folder provides:

- Clear separation from production code
- Easy reference for AI agents
- Versioned documentation of design decisions
- Implementation roadmaps and progress tracking
